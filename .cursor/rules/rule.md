你正在维护一个“橘源通”微信小程序原生项目。

技术约束：
- 使用微信小程序原生语法：WXML / WXSS / JS / JSON
- UI 组件优先使用 Vant Weapp
- 图表优先使用 echarts-for-weixin
- 不允许使用浏览器 DOM/BOM API
- 样式单位优先使用 rpx
- 页面逻辑尽量瘦，接口请求统一放在 services 目录

- 修改页面时，必须同时检查 .wxml / .wxss / .js / .json / app.json
- 所有新页面都要补路由
- 所有接口都要统一通过 request.js 请求
- 输出代码要可直接放进微信小程序项目中运行

- 项目名称：橘源通小程序
- 面向柑橘种植户、收购商、消费者
